# Auto Photo Studio (APS): The Foundry (Darkroom Edition)

![APS Cover](https://via.placeholder.com/1200x500/0a0a0a/ffffff?text=APS+The+Auto+Photo+Studio)

## 项目定位 (What is APS?)
APS 绝不是市面充斥的低劣“AI一键修图”套壳软件，而是一套专为影视从业者与顶配修图师打造的**【影视级全自动工业并行人像修炼管线】**。它彻底剥离了复杂玄学的人工点选步骤，依托 Google 顶级的 MediaPipe 物理切割技术与工业界封装死的极秘节点流，实现 0 废片级的高质感批量定妆。

我们针对 Apple Silicon (M1/M2/M3/M4) 设备进行了极致的 GPU (MPS) 深度优化。无论是深山老林的片场还是无网断联的地下影棚，打开系统，APS Darkroom 这套极纯粹的系统就能即刻吞下数百张生图、自动踢掉糊废件，精准除胶降噪，为您无情流转产出一流水准成片。

## 🎬 核心功能引擎链路

### 1. 废片过滤器 (The Sorter)
系统原生集成了一组极其稳固的 OpenCV Laplacian Variance（拉普拉斯锐度方差阵列）。对于因快门慢速或离焦导致的糊图废件，我们设置了 `100.0` 强力阈值。它们将毫无情面地被扫入垃圾池（🗑️），连进入 GPU 的资格都没有，极限捍卫出图纯净度！

### 2. 面网精细剥皮器 (The Mapper)
丢掉不可靠的视觉蒙版脑补。APS 直接调用极底层的 `MediaPipe FaceMesh` 库。
- 提取多达 **478 个特征骨骼点**的精准三维算力网。
- 以微厘级别的精度避开眼睛、唇形、眉毛，极速剔除所有不需要降噪的人体特征面，拉下一张专供毛孔降噪修复的高清黑白**物理皮肤蒙版 (Skin Mask)**！

### 3. 无头定调铸造炉 (The Forge)
您不会看到眼花缭乱的连线节点。通过 `forge.py` 后端调度，APS 将包含 **14个核心节点** 的工作载荷压缩并注入至本地被静默调用的 ComfyUI 服务中。
- 最冷酷的防翻车锁：我们的 `denoise` 值犹如保险栓般死死卡在 `0.25 - 0.35` 间，强制杜绝任何市面上 AI 修图产生的粗暴硅胶塑胶感，将高清的皮肤质感焊透进底片深层！

## 🌑 前端：APS Darkroom (暗房中枢面板)
我们摒弃了五颜六色的商业圆角，为您带来极致极客与赛博美学的操控台：
- **纯粹黑洞视觉**：纯粹底色 `#0a0a0a`、极致 `0px` 全角，以及精密的 `1px` 方形实线框。
- **纯键盘流交互**：修图师狂喜。将鼠标焦点指向中间屏幕，按住 `Space (空格)` ，当前图瞬间秒切为您相机里的原片；按下 `M` ，帅哥美女瞬间变成骨骼暴露的面部与全包围蒙版。
- **限位修图轨**：极致克制的阻尼垂直修图轨，拒绝无效拖拉，只有高级与废品的微差调音线！

## 🚀 极客本地母舰部署指南 (Mac M系发车向导)

如果你手中拥有一台 **MacBook Air / Studio / Pro (M1-M4芯片)**，您只需在键盘终端中按照以下三步拉起，这台电脑就会变成一台顶级的本地断网修图兵工厂！

### 第一步：获取属于您的金库代码
打开终端 (Terminal)，执行克隆本代码，并进入文件夹：
```bash
git clone https://github.com/JclawJJ/AutoPhotoStudio-Darkroom.git
cd AutoPhotoStudio-Darkroom
```

### 第二步：一键神级大基建部署 (The Setup)
我们为您内置了【无人值守级自动装载基建雷达】。它专门针对 Mac 架构底层加速优化，会自动为您抓取几十G的大模型、视觉库和必要节点：
```bash
# 给予脚本最高权限
chmod +x setup_comfy_aps.sh
# 执行启动，然后去睡觉或喝两杯咖啡
./setup_comfy_aps.sh
```

### 第三步：双核点火启动 💥
基建装载就绪后，我们需要在终端中分别打开两个面板。
**1. 开启雷达后台 (启动 Python 物理算力引擎)**：
*(在工程根目录执行)*
```bash
# 激活独立环境
source venv/bin/activate
# 点火后台引擎 (端口将开启于 8000)
python3 aps_server.py
```

**2. 开启您的极客暗房前台 (启动 Next.js 前端)**：
*(打开一个全新终端窗口，或在原界面的新 Tab 中执行)*
```bash
cd aps-darkroom
# 初始化前端库
npm install
# 启动前台 UI
npm run dev
```

现在，打开您的浏览器，输入 `http://localhost:3000`。
一个极其冷酷的修片终端已立于您的面前。把相机 SD 卡里的素材尽情向左侧面板拖拽吧！它开始吐图了。
